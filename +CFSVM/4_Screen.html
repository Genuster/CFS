Screen package#

CheckFrame class#

class CFSVM.Element.Screen.CheckFrame(parameters)#

Handling checkframes on the screen.

Derived from SpatialElement.

Constructor Summary
CheckFrame(parameters)#
Parameters:

  • checker_length – checker_length

  • checker_width – checker_width

  • hex_colors – color_codes

Property Summary
checker_length#

Int - length in pixels of every checker rectangle in the checkframe.

checker_width#

Int - width in pixels of every checker rectangle in the checkframe.

color_codes#

Cell array of chars contating HEX color codes e.g. {‘#0072BD’, ‘#D95319’, ‘#EDB120’, ‘#7E2F8E’} Red-Green {‘#00FF00’, ‘#FF0000’} White-Black {‘#FFFFFF’, ‘#000000’}

colors#

A matrix of size (3,N) for N=number of checkers

rects#

A matrix of size (4,N) for N=number of checkers.

CFSVM.Element.Screen.initiate(obj, screenfield)#

Initiates checkframe for a screenfield by generating rects and colors matrices.

Parameters:

screen – ScreenField object.

CFSVM.Element.Screen.rects_and_colors(obj, screen_rect)#

Calculates rectangles [x0,y0,x1,y1] and their colors for a checkframe.

Parameters:

screen_rect – [x0, y0, x1, y1] array with pixel positions.

Returns:

two matrices:

  • 4xN matrix describing rectangle of every checker.

  • 3xN matrix describing RGB color of every checker.

Return type:

[4xN matrix, 3xN matrix]

CFSVM.Element.Screen.reset(obj)#

Empties rect and color props.

fields#

Cell array of ScreenField objects.

frame_rate#

Float for display refresh rate.

Float for time between two frames ≈ 1/frame_rate.

Int for number of pixels shifted on keypress while adjusting screens.
